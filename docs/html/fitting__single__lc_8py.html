Definition which carries out the procedure for fitting a single wavelength.

This follows the steps of: initializing a TransitModel object with correct data for that wavelength, initializing an mcmc object and running it, saving the results of mcmc run, and making walker and triangle plots for that mcmc run. input_param_dic A dictionary of input parameter values LC_dic A light curve dictionary containing the wavelength data wl_id The key for the wavelength to be fit
