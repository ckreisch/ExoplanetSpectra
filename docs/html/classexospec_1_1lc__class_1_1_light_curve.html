exospec.lc_class.LightCurve Class Reference

List all the light curve files in the indicated folder if the user decided to use another wavelength resolution than the one provided with the files, the code compute the resulting number of new wavelength to be used loop over the light curve files to get their file names and original wavelengths if the user decided to use a new wavelength resolution, the code computes the new wavelength to be used instantiate the new lc objects (with new wave_bin_size) Warning: The code is not able yet to handle situation where files_num is not proportional to wave_bin_size.
